# Backend

Guides de développement pour le back-end du projet.

## Commandes utiles
- Lancer le service d'authentification : ./mvnw spring-boot:run
- Construire les services : ./mvnw clean package
- Vérifier les tests : ./mvnw test

## Conventions
- Respecter la structure par microservice.
- Préférer des services modulaires et bien séparés.
- Documenter les changements majeurs dans le code.
