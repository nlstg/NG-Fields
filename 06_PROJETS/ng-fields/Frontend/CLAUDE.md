# Frontend

Guides de développement pour le front-end du projet.

## Commandes utiles
- Installer les dépendances : npm install
- Démarrer le serveur de développement : npm start
- Construire le projet : npm run build

## Conventions
- Utiliser Angular/Next.js selon le sous-projet concerné.
- Garder le code propre et commenté si nécessaire.
- Vérifier les dépendances avant d'ajouter de nouvelles bibliothèques.
