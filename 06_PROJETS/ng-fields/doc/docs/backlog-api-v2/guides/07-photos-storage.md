## Endpoints

| Méthode | Endpoint | Description |
|---------|----------|-------------|
| POST | `/api/interventions/{id}/photos` | Upload photo (multipart) |
| GET | `/api/interventions/{id}/photos` | Liste des photos |
| DELETE | `/api/interventions/{id}/photos/{photoId}` | Supprimer une photo |
